# 作品信息摘要

---

## 项目名称

武汉外国语学校智能学习平台（AI Learning Platform）

## 版本

v2.0.0

## 项目简介

基于AI的智能学习平台，面向武汉外国语学校师生，提供AI智能导学、知识库管理、语义检索和学习进度追踪等功能。平台集成了多家主流AI大模型厂商，支持文档上传与智能结构化处理，通过向量嵌入技术实现语义级知识检索，帮助学生高效学习和掌握知识。

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | Vue 3 + Element Plus + Vite + Pinia + Vue Router |
| 后端 | Python FastAPI + SQLAlchemy + Pydantic |
| 数据库 | MySQL 8.0（utf8mb4） |
| AI模型 | SiliconFlow / OpenAI / Anthropic / 智谱AI / 通义千问 / ModelScope / Ollama / 自定义API |
| 向量搜索 | SiliconFlow Embedding API（BAAI/bge-large-zh-v1.5，1024维） |
| 认证 | JWT（HS256）+ bcrypt + RBAC角色权限（student/teacher/admin） |
| 部署 | Docker Compose（多阶段构建，Nginx反向代理） |

## 核心功能

### 1. AI智能导学

&emsp;**智能聊天**：与AI助手实时对话，支持流式SSE响应，可切换多家AI厂商和模型
 &emsp;**知识交互**：基于已上传文档的知识点进行交互学习
 &emsp;**智能检索**：支持关键词搜索、语义搜索、混合搜索（RRF融合）三种模式

### 2. 知识库管理

&emsp;**文档上传**：支持PDF、DOCX、TXT、XLSX、MD格式，最大50MB
&emsp;**知识结构化**：AI增强版+规则版双模式处理，自动提取大纲和知识点
&emsp;**向量索引**：文档自动分块并生成向量嵌入，支持语义级检索
&emsp;**知识库管理**：文档的增删查改，PDF在线预览

### 3. 用户与权限

&emsp;**用户认证**：注册、登录、JWT令牌管理
&emsp;**角色权限**：学生、教师、管理员三级RBAC权限控制
&emsp;**个人中心**：资料修改、密码修改

### 4. 学习进度追踪

&emsp;**文档进度**：阅读进度百分比、阅读时长统计
&emsp;**知识点掌握度**：0~1级别掌握度评估、复习次数记录
&emsp;**学习会话**：学习时段记录，统计学习时长

### 5. 系统管理

&emsp;**AI配置**：厂商切换、模型选择、API密钥管理
&emsp;**自定义模型**：添加兼容OpenAI格式的自定义API模型
&emsp;**用户管理**：管理员可管理用户角色和启用状态

## 项目结构

```
ailearning/
├── frontend/                    # 前端（Vue 3 SPA）
│   ├── src/pages/              # 页面组件（AI导学/智能学习/进度/设置）
│   ├── src/stores/             # Pinia状态管理
│   ├── src/utils/api.js        # 统一API调用封装
│   └── nginx.conf              # 生产环境Nginx配置
├── backend/                     # 后端（FastAPI）
│   ├── main.py                 # API路由定义（全部端点）
│   ├── core/                   # 核心模块（数据库/认证/配置/流处理）
│   ├── models/                 # ORM模型（9张数据表）
│   ├── services/               # 业务服务（文档处理/知识结构化/向量嵌入/混合检索）
│   └── ai_services/            # 多厂商AI服务实现（工厂模式）
└── docker-compose.yml          # Docker编排
```

## 数据模型

| 表名 | 说明 |
|------|------|
| users | 用户表（用户名、密码、角色、状态） |
| documents | 文档表（文件名、类型、内容、路径） |
| document_chunks | 文档分块表（分块文本、字符位置） |
| document_structures | 文档结构表（大纲JSON、知识点JSON） |
| embedding_vectors | 向量表（文本、嵌入向量） |
| chat_sessions / chat_messages | 聊天会话与消息 |
| learning_progress | 学习进度（百分比、时长、状态） |
| knowledge_mastery | 知识掌握度（掌握级别、复习次数） |
| study_sessions | 学习会话（开始/结束时间、时长） |

## 部署方式

- **Docker一键部署**（推荐）：`docker compose up -d --build`，包含MySQL + 后端 + 前端/Nginx
- **本地开发部署**：MySQL + Python后端 + npm前端，Vite代理API请求

## 支持的AI厂商

| 厂商 | 需要API Key | 说明 |
|------|------------|------|
| ModelScope | 否（内置） | 免费模型（MiniMax-M2.5） |
| Ollama | 否 | 本地部署模型 |
| SiliconFlow | 是 | 多种开源模型，有免费额度 |
| OpenAI | 是 | GPT系列 |
| Anthropic | 是 | Claude系列 |
| 智谱AI | 是 | GLM系列 |
| 通义千问 | 是 | Qwen系列 |
| 自定义API | 是 | 兼容OpenAI格式的任意API |
