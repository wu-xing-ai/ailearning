## 用户认证

### 用户注册

```
POST /api/auth/register
Content-Type: application/json
```

**请求参数:**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| username | string | 是 | 用户名（2-50字符） |
| password | string | 是 | 密码（至少4位） |
| display_name | string | 否 | 显示名称 |
| email | string | 否 | 邮箱 |
| role | string | 否 | 角色，默认student，可选：student/teacher/admin |

> 第一个注册的用户自动成为admin。

**响应示例:**
```json
{
  "id": "uuid-string",
  "username": "student1",
  "display_name": "学生1",
  "role": "student",
  "token": "eyJhbGciOiJIUzI1NiIs..."
}
```

---

## 文档管理

### 上传文档

```
POST /api/documents
Content-Type: multipart/form-data
```

**请求参数:**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| file | File | 是 | 文件（PDF/DOCX/TXT/XLSX/MD），最大50MB |

**响应示例:**
```json
{
  "id": "uuid-string",
  "status": "uploaded",
  "filename": "document.pdf",
  "file_type": "PDF",
  "file_path": "/path/to/knowledge_base/{id}/document.pdf"
}
```

> 文件上传后会在后台异步提取文本内容。

---

### 获取文档列表

```
GET /api/documents
```

**响应示例:**
```json
[
  {
    "id": "uuid-string",
    "filename": "document.pdf",
    "file_type": "PDF",
    "processed": false,
    "created_at": "2026-03-21T12:00:00"
  }
]
```

---

### 搜索文档

```
GET /api/documents/search?q=关键词
```

**请求参数:**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| q | string | 是 | 搜索关键词（匹配文件名和内容） |

**响应示例:**
```json
[
  {
    "id": "uuid-string",
    "filename": "document.pdf",
    "file_type": "PDF",
    "snippet": "...关键词相关内容...",
    "processed": false,
    "created_at": "2026-03-21T12:00:00"
  }
]
```

---

### 删除文档

```
DELETE /api/documents/{doc_id}
```

**响应示例:**
```json
{ "status": "deleted", "id": "uuid-string" }
```

> 同时删除文件、分块、结构化数据和嵌入向量。

---

### 获取文档预览

```
GET /api/preview?doc_id={doc_id}
```

**请求参数:**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| doc_id | string | 是 | 文档ID |

## AI聊天

### 与AI对话

```
POST /api/ai/chat
Content-Type: application/json
```

**请求参数:**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| prompt | string | 是 | 用户输入 |
| stream | boolean | 否 | 是否流式响应，默认false |
| modelConfig | object | 否 | 模型配置 |

**modelConfig 结构:**
```json
{
  "type": "custom",
  "name": "glm-5",
  "apiKey": "sk-xxx",
  "apiUrl": "https://..."
}
```

| 字段 | 说明 |
|------|------|
| type | 厂商：ollama/openai/anthropic/zhipu/qwen/siliconflow/modelscope/custom |
| name | 模型名称 |
| apiKey | API密钥（custom和需要密钥的厂商必填） |
| apiUrl | API地址（custom必填） |

**非流式响应:**
```json
{
  "response": "AI回复内容...",
  "model": "glm-5",
  "provider": "custom",
  "timestamp": "2026-03-21T12:00:00.000000"
}
```

**流式响应（SSE）:**
```
data: {"content": "AI"}
data: {"content": "回复"}
data: {"content": "的内容"}
data: {"done": true}
```

## 语义搜索与向量嵌入

### 语义/混合搜索

```
GET /api/search/semantic?q={query}&mode={mode}&top_k={top_k}&doc_ids={doc_ids}
```

**请求参数:**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| q | string | 是 | 搜索关键词 |
| mode | string | 否 | 搜索模式：keyword/semantic/hybrid，默认hybrid |
| top_k | int | 否 | 返回数量，默认10 |
| doc_ids | string | 否 | 文档ID列表，逗号分隔 |

**响应示例:**
```json
{
  "results": [
    {
      "chunk_index": 0,
      "text": "匹配的文本内容...",
      "score": 0.85,
      "document_id": "uuid-string",
      "filename": "document.pdf"
    }
  ],
  "mode": "hybrid",
  "total": 5
}
```

## 错误响应

所有接口出错时返回统一格式：

```json
{ "detail": "错误描述信息" }
```

**常见HTTP状态码:**

| 状态码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未认证或认证失败 |
| 403 | 权限不足 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |

---

## 认证说明

- 认证方式：JWT Bearer Token
- Token 有效期：24小时
- 算法：HS256
- 请求头：`Authorization: Bearer <token>`
