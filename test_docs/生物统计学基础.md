# 生物统计学基础

## 第1章 概述

### 1.1 什么是生物统计学

生物统计学是将统计学方法应用于生物学和医学领域的一门交叉学科。它结合了数学概率论、生物学实验设计和数据分析方法，为生命科学研究提供量化分析工具。

定义：生物统计学是运用数理统计的原理和方法，分析和解释生物学数据的一门学科。

### 1.2 生物统计学的发展历史

生物统计学起源于19世纪末，由弗朗西斯·高尔顿（Francis Galton）和卡尔·皮尔逊（Karl Pearson）等科学家奠基。高尔顿在研究遗传学时提出了回归分析的概念，这一概念后来成为统计学中最核心的方法之一。

注意：生物统计学的核心在于将数学的严谨性与生物学的复杂性相结合，通过概率模型理解生命现象中的不确定性。

## 第2章 概率论基础

### 2.1 随机事件与概率

在生物学实验中，很多现象具有随机性。例如，基因的遗传服从孟德尔定律，但具体每个后代表现出哪种性状是随机的。

- 样本空间：所有可能结果的集合
- 随机事件：样本空间的子集
- 概率：事件发生的可能性度量，取值范围 [0, 1]

### 2.2 条件概率与贝叶斯定理

条件概率在医学诊断中有重要应用。贝叶斯定理描述了在已知某些条件下，事件发生概率的更新方式。

公式：P(A|B) = P(B|A) × P(A) / P(B)

适用场景：疾病诊断、基因检测风险评估、药物反应预测。

### 2.3 常见概率分布

- 二项分布：描述n次独立实验中成功次数的分布，如n次基因检测中阳性结果的数量
- 正态分布：自然界中最常见的分布，如身高、血压等生理指标
- 泊松分布：描述单位时间内稀有事件发生次数，如突变事件

## 第3章 实验设计与数据收集

### 3.1 实验设计原则

生物统计学强调实验设计的重要性，好的设计可以减少偏差、提高效率。

- 随机化：将实验对象随机分配到各处理组
- 重复：每个处理至少有两个以上的实验单元
- 对照：设置对照组以消除混杂因素的影响
- 区组化：将相似的实验单元放在同一区组

### 3.2 抽样方法

在生态学研究中，常用的抽样方法包括：

- 简单随机抽样
- 分层抽样：按生境类型分层
- 整群抽样：以群落为基本单位
- 系统抽样：等距布设样方

## 第4章 描述性统计

### 4.1 集中趋势

- 均值：所有观测值的算术平均
- 中位数：将数据排序后位于中间的值
- 众数：出现频率最高的值

在生物数据中，中位数常比均值更能代表典型值，因为生物数据常常存在异常值（如个别极端体型）。

### 4.2 变异性度量

- 方差：衡量数据偏离均值的程度
- 标准差：方差的平方根，与原始数据同单位
- 变异系数：标准差/均值，用于比较不同量级数据的变异程度

## 第5章 假设检验

### 5.1 假设检验的基本思想

假设检验是生物统计学的核心方法之一。其基本思想是：

1. 建立零假设 H₀ 和备择假设 H₁
2. 选择检验统计量
3. 计算P值
4. 根据显著性水平α做出统计推断

### 5.2 t检验

t检验用于比较两组样本均值的差异。

- 单样本t检验：比较样本均值与已知总体均值
- 独立样本t检验：比较两组独立样本的均值
- 配对t检验：比较配对样本的差异

适用场景：比较两种药物的疗效差异、不同肥料对作物产量的影响。

### 5.3 方差分析（ANOVA）

当需要比较三组及以上样本时，使用方差分析。

- 单因素方差分析：一个自变量
- 双因素方差分析：两个自变量
- 重复测量方差分析：同一对象多次测量

### 5.4 卡方检验

卡方检验用于分析分类变量之间的关联性。

- 拟合优度检验：检验观测频率是否符合理论比例
- 独立性检验：检验两个分类变量是否独立

适用场景：基因型频率是否符合哈迪-温伯格平衡。

## 第6章 回归与相关

### 6.1 线性回归

线性回归分析两个连续变量之间的线性关系。

方程：Y = β₀ + β₁X + ε

其中：
- Y 是因变量（响应变量）
- X 是自变量（解释变量）
- β₀ 是截距
- β₁ 是斜率
- ε 是随机误差

### 6.2 相关系数

皮尔逊相关系数 r 衡量两个变量线性相关的程度和方向。

- r > 0：正相关
- r < 0：负相关
- r = 0：无线性相关

### 6.3 多元回归

当有多个自变量影响因变量时，使用多元回归分析。在生态学中，常用多元回归研究温度、湿度、光照等多个环境因子对物种多样性的影响。

## 第7章 交叉学科应用

### 7.1 流行病学中的统计方法

流行病学是生物统计学最重要的应用领域之一。

- 队列研究：追踪暴露组和对照组的发病情况
- 病例对照研究：比较病例组和对照组的暴露史
- 横断面研究：在特定时间点调查疾病的分布

### 7.2 遗传学中的统计方法

- 数量遗传学：利用统计学方法研究复杂性状的遗传规律
- 连锁分析：利用统计方法定位致病基因
- GWAS（全基因组关联分析）：在全基因组范围内寻找与性状相关的遗传变异

### 7.3 生态学中的统计方法

- 物种多样性指数：Shannon指数、Simpson指数
- 种群动态模型：利用差分方程和统计估计描述种群变化
- 群落分析：聚类分析、主成分分析（PCA）、非度量多维标度（NMDS）
